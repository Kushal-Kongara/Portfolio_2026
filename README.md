# Portfolio 2026

A modern, high-aesthetic, and interactive portfolio designed for 2026. This project showcases a blend of professional experience, technical expertise, and creative design using a "sketch-style" and "folder-based" UI.

## 🚀 Built With
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Interactivity**: Vapi.ai (Voice AI), React Icons
- **Language**: TypeScript

---

## 📂 Section Breakdown

### 1. Hero Section (The Hook)
*   **What I've done**: Created a bold entrance featuring a giant typography landing, an interactive social bar, and a **Voice AI Agent** that users can literally talk to.
*   **Concepts used**: 
    *   **AI Integration**: Integrated `Vapi.ai` for a live voice-conversational experience.
    *   **SVG Filters**: Custom "pencil-sketch" filters for a hand-drawn social icon aesthetic.
    *   **Spring Physics**: Fluid, spring-based entrance animations for the character graphics and text.

### 2. About Me (The Person)
*   **What I've done**: Designed a dual-layered section featuring a "Creative Studio" ID Badge (lanyard style) and a Polaroid-style photo grid of life moments/hackathons.
*   **Concepts used**:
    *   **Scroll-Linked Motion**: Used `useScroll` and `useTransform` to make the lanyard badge sway naturally as the user scrolls.
    *   **Bento Grid / Masonry**: A responsive photo layout that adapts based on the story's importance.
    *   **Retro UI**: Windows-style containers to house biological details.

### 3. Tech Stack (The Tools)
*   **What I've done**: Visualized technical skills using a scattered "graph paper" layout that mimics a developer's real-world desk/notebook.
*   **Concepts used**:
    *   **Glassmorphism**: Backdrop blur effects on tech icons for a premium feel.
    *   **Handwritten Aesthetics**: Specialized fonts and "highlighter" effects to emphasize tool descriptions.
    *   **Micro-interactions**: Subtle hover rotations and scale effects on each tool card.

### 4. My Journey (The Experience)
*   **What I've done**: Reimagined the traditional resume timeline as a series of "Lined-Paper Folders" decorated with custom SVG landmarks (like the Golden Gate Bridge or Charminar) based on location.
*   **Concepts used**:
    *   **SVG Art**: Custom-coded vector illustrations for location-based storytelling.
    *   **Paper Textures**: Lined-paper background gradients and transparent textures for a tactile feel.
    *   **Z-Index Layering**: "Peeking" decorations that sit behind or on top of folders to create depth.

### 5. Projects (The Work)
*   **What I've done**: Showcased software projects inside "Dark Mode" folders that reveal live links and GitHub source code upon interaction.
*   **Concepts used**:
    *   **Clip-Path Geometry**: Used complex CSS clip-paths to create a realistic folder "flap" effect.
    *   **Conditional Rendering**: Image-based cards vs. Gradient-based cards depending on the project type.
    *   **Backdrop Filters**: Blurred action buttons for better contrast over project images.

### 6. Contact (The Connection)
*   **What I've done**: A high-impact, minimalist contact section using a yellow-on-black "SWAG" color palette, simplified for the best user experience.
*   **Concepts used**:
    *   **Minimalist Form Design**: Clean border-bottom inputs without traditional boxes.
    *   **Typography Hierarchy**: Heavy, uppercase headers (`Impact` font) to drive a strong branding message.
    *   **Integrated Footer**: Merged navigation links directly into the contact flow.
